# Exploring Calculus with Maple

## LaTeX version

Files for the third edition of the lab manual are contained in the `latex` folder and can be compiled using PdfLaTeX. The book uses the `tufte-book` class.

## PreTeXt version

The `source` folder contains the necessary files for compiling in PreTeXt and output is generated in the `output` folder. To generate the output from a command line, use the command:
    pretext build web
The resulting html files can be viewed from a browser.
